¿Cómo genero un parte de visita?

Para poder generar un parte de visita, tengo que crear previamente una visita.

Una vez activada la tarea, podré imprimir el parte de visita desde el enlace que aparece en «Próximas actividades»

Se genera un Word, por lo que podré hacer modificaciones antes de imprimir.

Se puede acceder desde diferentes apartados…
