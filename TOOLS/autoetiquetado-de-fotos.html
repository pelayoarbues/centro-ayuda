Autoetiquetado de fotos

Esta funcionalidad permite identificar y etiquetar todos nuestros inmuebles de forma automática.

Entra en Inmuebles > Edición > Multimedia

De este modo cada vez que editemos un inmueble y vayamos al apartado multimedia, al subir nuestras imágenes , el programa se encarga de etiquetar cada una de las fotografías (salón, dormitorio, cocina, terraza, etc).

El proceso es el siguiente, al añadir las fotos para el inmueble el programa muestra

 

A medida que se van autoetiquetando las fotos aparece el contador

 

Después de unos segundos el programa nos avisa del número de fotos etiquetadas y muestra el mensaje con un aviso en color naranja si sólo ha sido capaz de etiquetar menos de 10 imágenes

 

Si el programa etiqueta todas las imágenes que hemos subido, es decir el 100%, nos muestra un aviso en color verde

 

El etiquetado de imágenes de nuestros inmuebles siempre se podrá revisar de forma manual, y si lo hacemos de este modo, prevalecerá sobre el etiquetado automático

Recuerda guardar siempre los cambios realizados
