Para potenciar la visibilidad de tus inmuebles con los productos multimedia disponibles en idealista, hemos creado el menu «Multimedia».

En «Multimedia» puedes gestionar las fotografías, vídeos, tour virtuales, virtual home staging, planos y otros que se mostrarán en el anuncio.

 

Entra en el menú Tus inmuebles > Editar > Multimedia

¿Donde puedo ver los productos multimedia para mis anuncios?

En este espacio verás los tipos de productos multimedia disponibles que puedes aplicar a cada uno de tus anuncios. Puedes ver qué producto multimedia tienes completado y cuántos elementos de cada tipo tienes publicados.

Puedes:

  • Subir tu propio elemento multimedia
  • Solicitar información sobre productos multimedia y contratar. Puedes ver los productos multimedia de idealista que no tienes contratados aún.
  • Crear multimedia (puedes descargar la plantilla de creación de planos)

1. Fotos

Puedes añadir fotos de hasta 32 mb de tamaño cada una (jpg, gif, png, jpeg). Máximo 180 fotos. Esto ayudará a aumentar la calidad de tus anuncios, no hay que olvidar que el sistema inteligente de idealista/tools posee un autoetiquetado de fotos automático para facilitar esta tarea.

2. Planos

 

 

Para subir un plano, debes únicamente subir la imagen y al etiquetarla señalar que se trata de un plano o incluso el propio programa lo etiqueta de forma automática.

Si lo que quieres es crear un plano, idealista/tools te indica paso a paso como utilizar  y descargarte la plantilla de planos.

 

3. Tour virtual

Si quieres mejorar y aumentar la calidad de tus anuncios, podrás añadir tus propios contenidos multimedia de tipo 3D y tour virtual 360º.

Una vez publicado el virtual tour, hecho por idealista, puedes visualizar desde idealista/tools cómo aparece ese virtual tour, lo podrás hacer en la sección de multimedia > tours virtuales 3d/360 >ver detalle:

La gran novedad es que desde este visor, puedes copiar el enlace del virtual tour para publicarlo en tu propia página web.

Para ello, tan sólo, debes ir al enlace de Publicar en tu página > copiar link para llevártelo a tu web Este link sólo se puede utilizar en tu página web, no en web de terceros.

4. Vídeos

Puedes incluir en tus anuncios 6 vídeos de hasta 600 megas, formatos válidos: avi, mov, wmv, mpeg, rm, mp4, flv, m2t, 3gp

5. VHS

Si además quieres diferenciarte y decorar virtualmente las estancias de un inmueble y ver cómo quedaría amueblado, iluminado y decorado adecuadamente, utiliza el Virtual Home Staging. Los resultados son increíbles!

6. Otros contenidos multimedia

Además podrás tener enlaces de Youtube o Vimeo.
